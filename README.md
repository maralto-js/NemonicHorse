## 🐴 Descrição

Transforma cavalos em companions estratégicos com habilidades únicas. Integração completa com MMOCore e ItemsAdder.

---

## ✨ Features

- 🐴 **4 Habilidades:** Impeto, Salto Heroico, Casco de Ferro, Esmagar
- ⚙️ **Configuração YAML:** Customize tudo sem alterar código
- 🔐 **Permissões:** Sistema granular via LuckPerms
- 📊 **Testado:** Rodando em servidor NemonicRP

---

## 🛠️ Instalação

1. Baixe o `.jar` compilado
2. Coloque em `plugins/`
3. Reinicie o servidor: `/reload confirm`
4. Edite `plugins/NemonicHorse/config.yml` conforme necessário

---

## 📖 Como Usar

/horse info        → Ver informações do seu cavalo
/horse skill dash  → Usar habilidade Impeto
/horse admin       → Comandos admin (OP)
